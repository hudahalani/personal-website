Reversi Game in C

University of Toronto Programming Course

Project Overview

Developed a complete implementation of the Reversi game in C for a university programming course. Created both player vs player and player vs computer modes with intelligent AI opponent. Competed against other students' implementations in class tournaments.

Key Features

  • Game Logic: Complete 8x8 board implementation with move validation
  • AI Opponent: Intelligent computer player with strategic decision making
  • User Interface: Terminal-based display with clear board visualization
  • Input Validation: Robust error handling for user input
  • Multiple Modes: Player vs Player, Player vs Computer, Computer vs Computer

Technical Implementation

  • Data Structures: 2D arrays for board representation and move tracking
  • Algorithms: Move generation, evaluation, and minimax with alpha-beta pruning
  • Memory Management: Efficient memory usage and optimization
  • Error Handling: Comprehensive input validation and error recovery

Competition Performance

Participated in class-wide tournaments where student implementations competed against each other. Continuously improved AI algorithms based on competitive results and performance analysis.
