SafeRoute: A Safety-First Navigation System

ECE297 - Software Design and Communication

Project Overview

Co-developed SafeRoute, a safety-first navigation platform for safer campus movement, especially for women. Built using C++ with a custom graphics engine, integrating OpenStreetMap data with advanced pathfinding algorithms (Dijkstra's, A*, Simulated Annealing).

Key Features

  • Safety-First Routing: Algorithms optimized for well-lit, populated routes
  • POI Safety Ratings: Google Places API integration for safety ratings
  • Emergency SOS: One-click location sharing with campus security
  • Accessibility: WCAG-compliant design with dark mode support
  • Real-time Updates: Live route updates and transit integration

Technical Implementation

  • Language: C++ with custom graphics engine
  • Mapping: OpenStreetMap API integration
  • Algorithms: Dijkstra's, A*, Simulated Annealing
  • Performance: Sub-100ms pan/zoom speeds
  • Emergency System: libcurl for location sharing

Results

Achieved System Usability Score of 91, demonstrating effective safety-first navigation. Successfully balanced performance, accessibility, and security features for campus use.
